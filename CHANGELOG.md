### Added

- Initial release: 29 components across primitives, layout, feedback, data
  display, and form patterns.
- ThemeProvider with light/dark support, `data-theme` attribute strategy.
- Design tokens as CSS custom properties (colors, spacing, typography,
  shadows, radii, motion).
- Full TypeScript definitions with strict types and named exports.
- Storybook documentation with a11y addon.
- Vitest unit tests with 80%+ coverage target.
